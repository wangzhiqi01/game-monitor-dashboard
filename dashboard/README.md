# 游戏监控系统 V1 原型

这是一个网页版静态原型，用来先确认：

- 页面结构
- 信息架构
- 字段设计
- 后续维护方式

## 目录
- `index.html`：主页面
- `styles.css`：样式
- `data.js`：示例数据
- `app.js`：页面渲染逻辑

## 先看什么
优先看三个问题：
1. 页面分区对不对
2. 你真正想盯的字段够不够
3. 哪些数据将来需要自动抓

## 本地打开
直接双击 `index.html` 即可在浏览器中查看。

## V1 现状
当前是静态原型，适合：
- 先确认产品结构
- 先做汇报演示
- 先决定要不要继续上 React / 后端 / 数据库

## 后续升级建议
- 用 React 重构
- 接 SQLite / PostgreSQL
- 接入自动抓取脚本
- 增加搜索、筛选、登录、多人协作能力
